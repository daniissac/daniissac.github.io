# Dani Issac — portfolio

The source for [daniissac.com](https://daniissac.com), a fast, accessible portfolio focused on network engineering, ThousandEyes, technical education, and open-source projects.

## Featured work

- [Internet in Motion](https://daniissac.com/internet-in-motion/) — interactive networking education
- [ConfigDiff](https://github.com/daniissac/configdiff) — semantic configuration comparison
- [whoson](https://github.com/daniissac/whoson) — subnet auditing and topology export
- [README Studio](https://daniissac.com/readme-studio/) — GitHub README authoring tool

## Local preview

The site is plain HTML and CSS, so it can be served with any static web server:

```bash
python -m http.server 8000
```

Open `http://localhost:8000`.

## Design goals

- Fast loading with no framework or runtime dependencies
- Accessible navigation, focus states, and reduced-motion support
- Responsive light and dark themes
- Clear paths from the portfolio to live projects and their source code
