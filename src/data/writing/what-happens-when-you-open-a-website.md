## Outline

1. The request starts before a packet leaves the device.
2. DNS turns a name into a destination.
3. The local network provides the first path outward.
4. Routers make independent forwarding decisions.
5. Transport and TLS establish a reliable, secure conversation.
6. The response travels back, often by a different route.
7. Observability tools reveal only part of the journey.
